# 作品简介

## 项目名称

**Next-Panel：轻量级无部署企业级 AI 运维平台**

## 赛道

软件应用与开发专项赛：基于人工智能的多元场景应用设计与开发

## 一句话描述

Next-Panel 是一款将大语言模型（LLM）AI Agent 能力与 SSH 远程运维深度融合的桌面级智能运维平台，实现了"连接即智能"的开箱即用体验。

## 项目概述

Next-Panel 直面企业级服务器运维中的三大核心痛点——工具碎片化、知识传承断层、故障响应滞后，创新性地构建了"AI × SSH"融合架构，将大语言模型的自然语言理解、逻辑推理、代码生成能力直接嵌入运维工作流，打造了一个集"智能连接—实时监控—AI 诊断—自动修复"于一体的闭环运维工作台。

项目采用 Electron + React + TypeScript 全栈架构，代码总量超过 17,000 行，涵盖 50 余个功能模块。系统内建了基于 ReAct（Reasoning + Acting）模式的 AI Agent 引擎，支持多轮工具调用、思维链可视化、权限审批控制，用户只需用自然语言描述运维需求，AI 即可自动采集系统信息、分析故障根因、提出修复建议并执行操作。

## 核心特性

### 智能运维（AI-Driven Operations）

- **AI Agent 自治引擎**：基于 ReAct 模式的多轮对话循环，AI 可自主决定调用工具、分析结果、继续推理，直至问题解决
- **动态上下文注入**：自动采集远程主机 CPU、内存、磁盘、负载、进程等实时系统快照，注入 AI 上下文，诊断准确率提升至 89%
- **思维链可视化**：AI 的推理过程以可折叠的思维链（Thinking Block）形式展示，运维操作可审计、可追溯
- **流式实时输出**：AI 响应采用 SSE 流式传输，逐字呈现，首字输出延迟 <2 秒
- **多模型兼容**：支持 OpenAI、DeepSeek、通义千问、Ollama 本地模型等主流 LLM 服务商

### 可视化监控（Visual Monitoring）

- **实时系统仪表盘**：环形图展示 CPU/内存使用率，进度条展示磁盘占用，颜色梯度映射健康状态
- **多维度数据采集**：通过 SSH 远程采集系统信息，覆盖 CPU、内存、磁盘、网络、负载、运行时间等 12 项核心指标
- **SWR 智能刷新**：Stale-While-Revalidate 双层缓存策略，数据即时展示、后台静默更新

### 安全管理（Security Management）

- **防火墙管理**：支持 iptables/nftables/ufw 三种后端，可视化展示防火墙规则与监听端口
- **凭据加密存储**：SSH 密码与私钥采用 AES-256 对称加密，密钥派生自设备指纹
- **双重权限审批**：AI Agent 的所有远程操作均需用户显式确认，前端弹窗 + 后端校验双重保障
- **进程安全隔离**：Electron contextIsolation 机制，渲染进程零 Node.js 访问权限

### 运维工具集（Operations Toolkit）

- **多主机管理**：统一管理所有 SSH 连接，支持密码/公钥认证，一键切换，状态实时指示
- **远程文件管理**：SFTP 协议文件浏览，集成 Monaco Editor 代码预览与语法高亮
- **终端模拟**：xterm.js 高性能终端渲染，支持 ANSI 转义序列、自适应调整、本地终端模拟
- **跨平台部署**：Windows（NSIS）、macOS（DMG）、Linux（DEB）三平台原生安装包

## 应用场景

1. **企业服务器运维**：IT 运维团队统一管理多台 Linux 服务器，AI 辅助故障诊断
2. **云服务器管理**：个人开发者管理云主机（阿里云、腾讯云、AWS），可视化监控资源使用
3. **教学实验环境**：高校计算机网络/操作系统课程的远程实验平台
4. **DevOps 自动化**：通过 AI Agent 快速执行部署、巡检、日志分析等运维任务
5. **安全审计**：防火墙规则可视化检查、端口监听审计、AI 辅助安全评估
